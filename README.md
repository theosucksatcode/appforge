# AppForge | SaaS Starter Template

## MCP Servers

This project ships with the Nuxt and Nuxt UI MCP servers declared inside the `.vscode` folder. These MCP servers have to be started manually before attempting any AI workflows so the agents can utilize these resources.

## Scripts

### Setup

Make sure to install dependencies:

```bash
pnpm install
```

### Development Server

Start the development server on `http://localhost:3000`:

```bash
pnpm dev
```

### Production

Build the application for production:

```bash
pnpm build
```

Locally preview production build:

```bash
pnpm preview
```

Check out the [deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.
